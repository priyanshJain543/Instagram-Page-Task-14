# 📸 3D Instagram-Style Heart Interaction

A high-fidelity UI component that replicates the iconic **Instagram double-tap** animation with a sophisticated **3D twist**. This project leverages modern CSS properties to create depth and motion on the Z-axis.

## ✨ Features

* **3D Perspective Engine**: Built using `perspective: 1000px` to create a realistic spatial environment.
* **Dynamic Z-Axis Motion**: The heart doesn't just scale; it moves toward the viewer using `translateZ(200px)`.
* **Physics-Based Animation**: Uses a custom `cubic-bezier(0.175, 0.885, 0.32, 1.275)` timing function for a bouncy, natural "pop" effect.
* **Modern Glassmorphism**: High-contrast drop shadows and text shadows create a floating 3D glass effect.
* **SCSS Optimized**: Fully modular design using variables for easy maintenance and scalability.



---

## 🚀 Technical Stack

* **Structure**: HTML5 with Semantic Elements.
* **Styling**: SCSS (Nesting, Variables, 3D Transforms).
* **Interaction**: Vanilla JavaScript (Event Loop & DOM Manipulation).
* **Icons**: Remix Icon Library.
